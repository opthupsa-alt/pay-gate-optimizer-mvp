القسم 11: المميزات والعيوب
================================================================================

### ⭐ المميزات بالعربية:
1. انتشار قوي ومتسارع داخل السعودية.  
2. يدعم مدى + المحافظ + BNPL + Apple Pay.  
3. دعم قوي للإشتراكات والمدفوعات المتكررة.  
4. واجهة برمجية واضحة وسهلة الدمج.  
5. دعم حقيقي للـ POS + SoftPOS.  

### ❗ العيوب بالعربية:
1. الأسعار غير شفافة — لازم تفاوض.  
2. بعض التجار اشتكوا من تباين جودة الدعم الفني.  
3. المستندات الرسمية محدودة في تفاصيل معينة (أقل من Stripe / Checkout).  

### ⭐ المميزات بالإنجليزية:
1. Solid API + SDK ecosystem.  
2. Good balance between affordability and enterprise readiness.  
3. Strong BNPL and wallet integrations.  

### ❗ العيوب بالإنجليزية:
1. Pricing is not public for KSA.  
2. Documentation still not as rich as Tier-1 providers (Stripe/Checkout).
