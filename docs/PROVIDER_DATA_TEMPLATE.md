# 📋 قالب بيانات بوابات الدفع - النموذج النهائي الرسمي

> **الإصدار:** 1.0  
> **التاريخ:** December 9, 2025  
> **الغرض:** نموذج موحد لتعبئة بيانات كل بوابة دفع تُستخدم في المقارنات

---

## 🎯 كيفية الاستخدام

1. انسخ القالب أدناه لكل بوابة
2. املأ جميع الحقول (اكتب "غير متاح" أو "0" إذا لا ينطبق)
3. أرفق رابط المصدر الرسمي لكل معلومة
4. سأقوم بتحويل البيانات مباشرة إلى قاعدة البيانات

---

## 📊 العوامل المؤثرة في المقارنة (الأوزان الحالية)

| العامل | الوزن | الوصف |
|--------|-------|-------|
| **التكلفة (cost)** | 30% | إجمالي التكلفة الشهرية المتوقعة |
| **التوافق (fit)** | 20% | مدى ملاءمة المزود للقطاع والاحتياجات |
| **العمليات (ops)** | 15% | كفاءة العمليات والدعم والتوثيق |
| **المخاطر (risk)** | 15% | مستوى المخاطر والاستقرار |
| **التقييمات (rating)** | 10% | تقييمات المستخدمين |
